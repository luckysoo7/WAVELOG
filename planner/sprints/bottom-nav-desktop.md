# Sprint: 하단 네비바 데스크톱 확장

## 목표
모바일 전용(`md:hidden`)이던 하단 날짜 네비바를 데스크톱까지 확장.
트랙리스트를 다 읽고 나서 자연스럽게 날짜를 넘길 수 있게 한다.

## 문제 정의
- 데스크톱: 하단 바 없음 → 트랙 20개 다 읽으면 갈 곳이 없음
- "전체 목록" 라벨: 누르면 `/bcamp`로 이동하는데, 이름이 동작을 설명 못 함
- 바가 `left:0`이라 데스크톱에서 사이드바(w-72 = 18rem) 아래까지 덮음

## 변경 범위

### 1. globals.css — `.date-nav-bar`
- md+ 에서 `left: 18rem` 추가 (사이드바 너비 맞춤)

### 2. PlaylistView.tsx
- `md:hidden` 제거 → 전체 노출
- 중앙 버튼: "전체 목록" → 프로그램 단축명 + 홈 아이콘 (배캠/별밤)

## 성공 기준
- [ ] 데스크톱: 하단 바가 사이드바 오른쪽부터 시작 (사이드바 침범 없음)
- [ ] 데스크톱: ← 최신날짜 | 배캠 홈 | 오래된날짜 → 정상 동작
- [ ] 모바일: 기존과 동일 동작
- [ ] 중앙 버튼 클릭 시 프로그램 허브(/bcamp 또는 /byulbam) 이동

- `web/src/app/globals.css`
- `web/src/components/PlaylistView.tsx`
