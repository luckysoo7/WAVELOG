## Q8. OAuth 토큰은?

현재 `match.py`가 `token.json`으로 YouTube API 씀. scope: `youtube`.
`videos.insert`에는 `youtube.upload` scope 필요 → **재인증 필요 가능성 있음.**

확인 방법: `token.json` 의 scope 필드 보면 됨.

추천: 기존 token.json scope 확인 → 부족하면 로컬에서 한 번 재인증 → 새 token GitHub Secrets 업데이트.
